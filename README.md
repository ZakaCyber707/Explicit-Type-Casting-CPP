ASCII Character Converter (Explicit Casting)


📖 Project Overview
This repository showcases the implementation of Explicit Type Conversion in C++. Unlike implicit casting, this program uses the robust static_cast operator to transform data types, ensuring type safety and code clarity.

⚙️ Technical Logic
The program performs a direct conversion between different data representations:
Character Capture: Accepts a char input from the user.
Explicit Conversion: Utilizes static_cast<int>(ch) to retrieve the underlying ASCII value.
Numerical Output: Prints the resulting integer to the console.


🛠️ Features
Type Safety: Uses modern C++ static_cast instead of old C-style casts for better compile-time checking.
Data Insight: Provides a simple way to visualize how computers store text as numerical data.



📋 Prerequisites
Ensure you have a C++ compiler installed:
Windows: MinGW or Visual Studio
macOS: Xcode Command Line Tools (g++)
